LLCDAO Bylaws & Guides

このサイトは、定款・各規程の本文と運用ガイドをブラウザで閲覧できる公開ビューです。左の「大カテゴリ」をクリックして展開し、読みたい文書を選択してください。

公開コメントの送り方

  • Issue で提案（推奨）: 新規 Issue を作成
  • Pull Request で提案: 差分を作成（対象ファイルを直接編集し、変更点を明確に記載）
  • フォーム（GitHub不慣れな方向け）: 変更提案フォーム

原則、毎週水曜を目途に内容を確認し、反映可否を判断・更新します。

収録内容

  • 定款・規程（最新版）: 定款 v2.0、DAO総会規程 v2.1、運営規程 v2.1、トークン規程 v2.2、用語集
  • 別紙: DAO総会別紙、共通運用、トークン別紙
  • ガイド: ガイド概要、立上げガイド（Setup）、運用ガイド（Operation）

詳細はリポジトリの README も参照ください。
