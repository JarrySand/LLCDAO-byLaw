## 🎯 このプロジェクトが解決する課題

日本国内で「DAO」を実社会の課題解決や市場創出に活用するには、契約主体・法定通貨の取り扱い・有限責任の適用など、多くの課題があります。これらの解決策として期待されるのが**法人制度の活用**です。

しかし、各DAOが独自に定款や規程を作成しようとすると、数十〜数百万円規模の法務コストが発生します。これは試行を重ねるうえでの大きな障壁です。

**本プロジェクトは、DAOの法人利用時の定款・規程の汎用的なひな形を提供することで：**

- 📉 **各DAOの法務コスト（限界費用）を限りなくゼロに近づける**
- ✅ **適法性の高い運用によりDAOの社会受容を促進する**

DAOの社会実装を推進する**公共財（Public Good）**として、成果物はオープンソースで公開しています。

## 🏗️ 設計原則

本ひな形は以下の原則に基づいて設計されています：

- **多元性** — 様々な思想やシステムを実現する土台となるよう、モジュール化
- **透明性** — ルールや情報を可能な限り公開し、参加者が納得して参加できる環境
- **操作可能性** — 法務に詳しくない方でもパラメータを調整しやすい別紙構成
- **責任所在の明確化** — 誰がどう責任を負うか、免責されるのはどんな場合かを明示
- **DAOによる意思決定の影響力の最大化** — 現行法で可能な範囲でDAO決議を法人決議に直結

詳細は [設計思想（context）](./docs/guide/context.md) をご覧ください。
