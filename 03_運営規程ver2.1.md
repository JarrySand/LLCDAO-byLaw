# 合同会社型DAO運営規程 ver2.1

## 第1章　業務執行体制

### 第1条（業務執行社員及び代表社員の定義）
本規程において、業務執行社員とは、本DAO定款における業務執行社員をいい、代表社員とは、本DAO定款第11条の選定方法に基づき選定される代表社員をいう。

### 第2条（業務執行社員の選任・権限）
1. 本DAOの業務執行社員は、DAO総会規程第6条に基づく重要決議を経て、社員の中から選出されるものとする。
2. 業務執行社員は、定款第19条に定める拒否権を行使する場合を除き、DAO総会の決議の本旨に従い本DAOの業務を執行する権限を有し、義務を負う。ただし、定款第18条に定める社員による否認決議があった場合にはこの限りではない。
3. 業務執行社員は、定款第18条に定める社員による否認決議及び定款第19条に定める拒否権が行使された場合に限り、、DAO総会により決議された事項の差し止め、及び実施中の事業の差し止めを行うことができる。

### 第3条（代表社員の選任・権限）
1. 代表社員は、DAO総会規程第6条に基づく重要決議により、業務執行社員の中から選出されるものとする。
2. 代表社員は、本DAOを代表する。
3. 署名担当代表社員は、DAO総会規程第6条に基づく通常決議により、代表社員の中から選出され、本DAOの代表署名・代表印の管理及びこれらの適切な使用につき本DAO及び社員に対して責任を負う。なお、署名担当代表社員が選出されていない場合は、すべての代表社員が本項の権利と義務を有するものとする。

### 第4条（業務執行社員会の運営）
1. 業務執行社員は、業務執行に関する決議を行うために、業務執行社員会を設置することができる。
2. 業務執行社員会における議決は、法令又は定款に別段の定めがある場合を除き、業務執行社員の過半数による議決をもって議案の承認とする。
3. 業務執行社員は止むを得ない事由がある場合を除き、業務執行社員会における投票に参加しなければならない。
4. 業務執行社員会は、必要がある場合に業務執行社員が招集し、開催する。
5. 業務執行社員会の議事については、議事録を作成し、全社員によって閲覧できるようにしなければならない。ただし、業務執行社員会は当該議事録に個人情報及び機密情報は記載しないものとする。（使用ツールは別紙「共通運用.md」を参照する。）

### 第5条（各担当社員の役割分担）
1. 財務担当業務執行社員は、DAO総会規程第6条に基づく通常決議により、業務執行社員の中から選出され、トレジャリー管理規程第3条に定める権限及び責任を負う。
2. 経理担当業務執行社員は、DAO総会規程第6条に基づく通常決議により、業務執行社員の中から選出され、トレジャリー管理規程第4条に定める権限及び責任を負う。
3. 前2項の担当社員が選出されない場合、代表社員がその職務を兼任するものとする。

## 第2章　業務執行プロセス

### 第6条（DAO総会決議の執行義務）
1. 業務執行社員は、DAO総会規程に基づき適法に成立した決議について、遅滞なくその執行に着手しなければならない。
2. 前項の決議の執行は、決議の内容及び重要度に応じ、当該決議において別段の定めがない限り、以下の期限内に着手するものとする。
      - 通常決議：決議成立後30日以内
   - 重要決議：決議成立後14日以内
   - その他の決議：決議成立後7日以内
3. 業務執行社員は、第2条第3項に定める場合を除き、決議内容に従い誠実に業務を執行しなければならない。

### 第7条（業務執行の再委任）
1. 業務執行社員は、DAO総会の委任決議に基づく業務執行を行うに際しては、以下の範囲で業務執行の権限を第三者に再委任することができる。
   - 日常的業務の執行（月額100万円以下の支出を伴う業務）
   - 定型的契約の締結（標準契約書に基づく契約）
   - 緊急時対応（事業継続に必要な応急措置）
2. 前項の委任を受けた者は、委任の範囲内で業務を執行し、その結果について業務執行社員に報告する義務を負う。
3. （本条における再委任においては、委任者は委任を受けた者に対して委任者と同様の責任を負わせるものとし、）委任者は委任を受けた者の業務の結果に責任を持つものとする。

### 第8条（執行状況の監督・報告）
1. 業務執行社員は、DAO総会決議の執行状況について、別段の定めがない限り、当該決議の執行が終了するまでの期間、以下の頻度で報告しなければならない。
      - 通常決議：四半期ごと
   - 重要決議：毎月
   - 委任決議：毎週
2. 前項の報告は、DAO総会規程第2条で定める議決権保有者（以下、議決権保有者という。）が閲覧可能な方法で行わなければならない。
3. 議決権保有者は、業務執行の状況について説明を求めることができ、業務執行社員は合理的な範囲内でこれに応じる義務を負う。

### 第9条（執行困難時の対応手続き）
1. 業務執行社員は、DAO総会の決議を執行することが以下の理由により困難又は不適切と判断する場合、遅滞なくその旨を議決権保有者に報告し、DAO総会規程第4条に基づく審議を求めることができる。
   - 法令違反のおそれが生じた場合
   - 本DAOに重大な損害を与えるおそれが生じた場合
   - 客観的に実行が不可能又は著しく困難な事由が生じた場合
2. 前項の報告には、執行困難の理由、想定される影響、及び代替案（ある場合）を含めなければならない。
3. DAO総会において執行困難が認められた場合、当該決議は修正又は取り消されるものとする。

## 第3章　情報開示・透明性

第10条（一般開示事項)
本DAOは、以下の事項をインターネット上で誰もがアクセス可能な公開情報として開示する。
1. 定款、DAO総会規程、運営規程、トレジャリー管理規程、トークン規程、DAO憲章、及びこれらに類する重要な規程類
2. トレジャリー資金を保有するパブリックアドレス
3. トークン規程で定める各種トークンのコントラクトアドレス
4. その他本DAOの運営にかかわるスマートコントラクトのコントラクトアドレス

（注）開示に用いる具体的なチャネル・周知・記録・保存方法は別紙「共通運用.md」を参照する。

### 第11条（事後開示事項）
業務執行社員は、本DAOに以下の事項が発生した場合には、遅滞なく議決権保有者に情報開示するものとする。
1. 本DAOの支払停止若しくは支払不能又は手形若しくは小切手の不渡り
2. 本DAOに対する、破産手続開始、会社更生手続開始、民事再生手続開始、特別清算開始、又はその他の倒産手続開始の申立て
3. 監督官庁による営業停止、営業許認可若しくは登録の取消処分、指導、又は調査
4. 災害その他の重大な損害のおそれのある事象の発生
5. 事業計画の実現に重大な影響を生ずるおそれのある事項

（注）当該開示の具体方法は別紙「共通運用.md」を参照する。

### 第12条（定期情報提供）
業務執行社員はは、以下の各号の資料を、当該各号に定める時期までに議決権保有者に情報開示する。
1. 各事業年度の計算書類：各事業年度終了後90日以内
2. 各トークンの発行状況：変更があった場合で議決権保有者の要求があったときは速やかに

（注）提供方法・記録・保存は別紙「共通運用.md」を参照する。

### 第13条（開示手続き・基準）
1. 前3条に定める情報開示は、以下の方法のいずれかにより行われることで完了したものとみなす。
   - DAO総会プラットフォームでの公開
   - 公式ウェブサイトでの掲載
   - その他議決権保有者が容易にアクセス可能な方法
2. 開示する情報には、個人情報及び機密情報を含めないものとする。ただし、法令により開示が義務付けられている場合はこの限りでない。
3. 議決権保有者から合理的な範囲内の情報開示の請求があった場合、業務に支障がない範囲で速やかに対応するものとする。
4. 具体的なチャネル設定・周知・記録・保存等の手続の詳細は別紙「共通運用.md」を参照する。

## 第4章　具体業務細則

### 第14条（具体業務の執行について）
1. 本DAOは、DAO総会規程第6条に基づく通常決議に基づき、具体業務に関する細則を設置することができる。
2. 前項の規定にかかわらず、業務執行社員は、DAO総会規程第9条で委任された範囲において、新たに細則を設置することができる。
3. 業務執行社員及び本規程第7条で再委任を受けた者は、前2項で定める細則に従わなければならない。

## 第5章　雑則

### 第15条（改廃）
この規程の改廃は、DAO総会規程第6条に基づく決議に従い行う。
