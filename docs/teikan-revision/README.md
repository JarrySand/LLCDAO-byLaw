## 2. 暫定結論一覧（17論点）

| 2 | 機能不全条項 | **「業務執行が一切行われない状態が2年以上継続」のみ**に限定（ver2.0の4類型から1類型のみ採用） | 修正採用 | ✅ 反映済 |

### 3.2 ガバナンス構造の変更

```
【ver2.0】
DAO総会決議 → 社員同意プロセス → （否認決議・拒否権）

【ver3.1】
DAO総会 ＋ 業務執行社員会の二層構造
├── 一部事項は両機関の決議が必要
└── 規程がない場合は定款のミニマムルールに従う
```

## 4. ver2.0からの主要変更点サマリー

### 4.1 削除された機能

| 項目 | ver2.0 | ver3.1 | 理由 |
|------|--------|--------|------|
| 社員同意プロセス（第18条） | あり | 削除 | 業務執行社員会方式に変更 |
| 否認決議制度（第18条第3項） | あり | 削除 | 同上 |
| 拒否権制度（第19条） | あり | 削除 | 同上 |
| 機能不全条項（4類型） | あり | 1類型のみ | シンプル化 |
| 暗号資産非該当明記 | あり | 削除 | 暗号資産であること自体は問題なし |
| 設立時資本金規定 | あり | 削除 | - |
| 本店所在場所規定 | あり | 削除 | - |

### 4.2 追加された機能

| 項目 | ver2.0 | ver3.1 | 理由 |
|------|--------|--------|------|
| 業務執行社員会（第5章全体） | なし | 追加 | macver型ガバナンス採用 |
| 代替トークン発行但書 | なし | 追加 | 緊急時対応 |
| 持分払戻し規定 | なし | 追加 | 社員保護・出口確保 |
| 議事録作成義務（定款明記） | なし | 追加 | 運用の確実性向上 |
| 規程不存在時のフォールバック規定 | なし | 各所に追加 | 設立初期の運用対応 |

### 4.3 変更された規定

| 項目 | ver2.0 | ver3.1 | 理由 |
|------|--------|--------|------|
| DAO総会権限 | 11項目（簡潔） | 18項目（詳細） | 権限範囲の明確化 |
| 解任方法 | DAO総会＋総社員過半数 | DAO総会のみ | シンプル化 |
| 過失要件 | 過失 | 重過失 | 業務執行社員のなり手確保 |
| 出資超過制限対象 | 業務執行社員以外のトークンホルダー | トークンホルダー全体＋実質要件 | 金商法対応の明確化 |
| 配当財産の種類 | 具体例なし | 日本円/暗号資産/トークンを明記 | DAO特性への対応 |
| 倒産等手続定義 | 包括規定 | 詳細列挙 | 明確性向上 |

## 5. 金融庁確認対象事項

以下の事項については、金融庁への確認が予定されています：

1. **持分払戻し**（論点11）
   - 現金等による出資に対して、暗号資産等で配当／払戻しがOKなのか
   - 特にステーブルコイン（JPYCなど電子決済手段）での払戻し可否

2. **配当財産の種類**（論点15）
   - 暗号資産やトークンでの配当の法的位置づけ
