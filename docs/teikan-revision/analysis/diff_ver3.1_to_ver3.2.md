## 設計思想の変更

```
【規程なし時】
DAO総会 ←──両機関決議──→ 業務執行社員会（第20-23条）
         （ブレーキ）

DAO総会 ──→ 業務執行社員の拒否権（第18条）──→ 執行
              （最低限のブレーキ）
              
※業務執行社員会（第20-23条）は【失効】
※規程で業務執行社員会を【再編可能】
```

## 2. 変更詳細

### 2.1 第12条（業務執行の決定）第1項

**変更内容**: 運営規程の制定主体を柔軟化

| ver3.1 | ver3.2 |
|--------|--------|
| 当会社の業務執行（トレジャリーその他資金の運用を含む。）は、**業務執行社員会において**別途定める運営規程（以下「運営規程」という。）に従うものとする。 | 当会社の業務執行（トレジャリーその他資金の運用を含む。）は、**DAO総会又は業務執行社員会において**別途定める運営規程（以下「運営規程」という。）に従うものとする。 |

**理由**: DAO総会規程が存在する場合、DAO総会が運営規程を制定できるようにするため。

### 2.2 第14条（DAO総会の設置）

| ver3.1 | ver3.2 |

| 当会社は、トークンホルダー全員で組織するDAO総会を置く。<br>（第2項なし） | 当会社は、トークンホルダー全員で組織するDAO総会を置く。<br>**DAO総会規程で規定されていない事項については、トークンホルダーの過半数の同意で決定するものとする。** |

### 2.3 第15条（DAO総会の権限）

**変更内容**: 

1. **柱書を全面修正**（規程有無による分岐を明確化）
2. **第1号として包括条項を追加**
3. **第14号（旧第13号）の運営規程変更権限を拡大**

#### 柱書の変更

| ver3.1 | ver3.2 |
|--------|--------|
| DAO総会は、以下の権限を有するものとし、DAO総会において別途定めるDAO総会規程（以下「DAO総会規程」という。）に従い、提案及び決議を行うものとする。**第19条**に基づき業務執行社員会の権限とされている事項については、DAO総会の決議に加えて、業務執行社員会の決議を要するものとする。 | DAO総会は、以下の権限を有するものとし、DAO総会において別途定めるDAO総会規程（以下「DAO総会規程」という。）に従い、提案及び決議を行うものとする。**DAO総会規程が定められていない場合は、第21条に基づき業務執行社員会の権限とされている事項については、DAO総会の決議に加えて、業務執行社員会の決議を要するものとする。DAO総会規程が定められている場合は、第18条に定める拒否権の適用を受けるものとする。** |

**理由**: 
- 規程なし時：業務執行社員会との両機関決議
- 規程あり時：拒否権の適用を受ける（業務執行社員会は失効）

#### 権限列挙の変更

| ver3.1 | ver3.2 |
|--------|--------|
| （第1号なし） | **当会社のガバナンス、ファイナンス及び業務執行に関する重要な事項の決定** |
| 第1号：社員が社員権トークン... | 第2号：社員が社員権トークン... |
| ... | ... |
| 第12号：DAO総会規程、トークン規程及び運営規程**（トレジャリー管理規程に限る。）**の変更 | 第14号：DAO総会規程、トークン規程及び**運営規程**の変更 |
| 第13号〜第18号 | 第15号〜第20号（番号のみ変更） |

**理由**: 
- 第1号追加：DAOが自己規定的に権限を解釈できるようにするための包括条項
- 第14号修正：DAO総会が運営規程を含むすべての規程を変更できるようにする
- 但書追加：規程がある場合に両機関決議を上書き可能にする

### 2.4 第16条（DAO総会の決議事項）【新設】

**変更内容**: 新規追加

```
第16条　（DAO総会の決議事項）
前条第1号によりDAO総会の決定を要する重要な事項は、DAO総会規程の定めによるものとする。
DAO総会規程に定めがない又はDAO総会規程が定められていない場合は、業務執行社員会の
過半数の同意で決定するものとする。
```

**理由**: 
- DAO総会規程で「重要な事項」を自己定義可能にする
- 規程がない場合は業務執行社員会が暫定的に決定（ブレーキ機能）
- ver2.0の第16条に相当

---

### 2.5 第18条（業務執行社員の拒否権）【新設】

**変更内容**: 新規追加

```
第18条　（業務執行社員の拒否権）
DAO総会規程が定められている場合、DAO総会の決議が次に定める事項に該当するときは、
業務執行社員の過半数による決議をもって、当該DAO総会の決議を差し止めることができる。
この場合、業務執行社員はトークンホルダーに対し、当該事項について必要な説明を
しなければならない。
(1)　法令に違反する場合
(2)　人の生命又は身体の保護のために必要がある場合
(3)　国の機関若しくは地方公共団体又はその委託を受けた者が法令の定める事務を遂行する
    ことに対して協力する必要がある場合等のやむにやまれない必要性がある場合
(4)　当会社の定める定款又は規則に違反する場合
当該拒否権の行使に関する決議は、前項各号への該当を示す事実の発覚から●日以内に
行われなければならない。
```

**理由**: 
- 規程存在時の最低限のブレーキとして機能
- ver2.0の第19条（拒否権）と同様の免責事項
- 規程なし時は適用されない（業務執行社員会との両機関決議がブレーキ役）

### 2.6 第20条（業務執行社員会の設置）※旧第18条

**変更内容**: 第2項として失効条項を追加

| ver3.1（第18条） | ver3.2（第20条） |
|-----------------|-----------------|
| 当会社は、業務執行社員で組織する業務執行社員会を置く。 | 当会社は、業務執行社員で組織する業務執行社員会を置く。<br>**ただし、DAO総会規程が定められている場合は、本章（第20条から第23条）の規定は適用しないものとする。この場合において、DAO総会規程で業務執行社員会に関する定めがあるときは、当該定めに従うものとする。** |

**理由**: 
- 規程存在時に業務執行社員会を【失効】させる
- 必要であれば規程で【再編】可能

### 2.8 第21条（業務執行社員会の権限）※旧第19条

**変更内容**: 

1. **第1号に但書を追加**（規程の制定・変更を除外）
2. **第9号「運営規程の変更」を削除**
3. **条文番号参照を修正**（第15条はそのまま）

※柱書は変更なし（但書は不要。第20条で失効するため）

#### 第1号の変更

| ver3.1（第19条） | ver3.2（第21条） |
|-----------------|-----------------|
| 当会社のガバナンス、ファイナンス及び業務執行に関する重要な事項の決定 | 当会社のガバナンス、ファイナンス及び業務執行に関する重要な事項の決定**（ただし、DAO総会規程、トークン規程及び運営規程の制定又は変更を除く。）** |

**理由**:
- **規程の制定・変更は業務執行社員会の「重要な事項」に含まれないことを明確化**
- 第22条（業務執行社員会の決議事項）で「重要な事項」を自己定義する際に、規程変更を含めることを防止
- DAOが規程を制定して主権を確立する道を保障

#### 権限列挙の変更

| ver3.1（第19条） | ver3.2（第21条） |
|-----------------|-----------------|
| 第1号〜第8号 | 第1号〜第8号（第1号に但書追加） |
| **第9号：運営規程の変更（トレジャリー管理規程は除く。）** | **削除** |
| 第10号〜第14号 | 第9号〜第13号（番号繰り上げ） |

**理由**: 
- 運営規程の変更権限をDAO総会に一本化

## 3. 変更の目的サマリー

### 設計思想の実現

```
【規程なし時】
DAO総会 ←──両機関決議──→ 業務執行社員会（第20-23条）
         （ブレーキ）

・定款のミニマムルールが適用
・業務執行社員会がセーフガードとして機能

【規程あり時】
DAO総会 ──→ 業務執行社員の拒否権（第18条）──→ 執行
              （最低限のブレーキ）

・業務執行社員会（第20-23条）は【失効】
・必要であれば規程で【再編】可能
・最低限のブレーキとして拒否権が有効化
```
