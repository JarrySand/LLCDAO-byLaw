# 2. 定常業務カレンダー（年間スケジュール）

DAOを維持するために「いつ、何をするか」のリストです。Googleカレンダー等に登録して、抜け漏れを防ぎましょう。

## 月次・四半期ルーチン

### 毎月（Monthly）
- [ ] **月次報告**（担当: 業務執行社員）
    - **時期**: 翌月5営業日以内
    - **目的**: 業務執行社員は、DAO総会から「運営」を委任されている立場です。委任された業務（予算執行や日々のタスク）が適切に行われているかを、委任者であるメンバーに報告・説明する義務があります（運営規程第8条）。
    - **報告内容**:
        1. **活動実績**: 「先月は何をしたか」（例: マーケティング施策の進捗、提携交渉の状況）
        2. **会計報告**: 「いくら使い、いくら残っているか」（例: ツール代〇〇円、残高〇〇円）
        3. **来月の予定**: 「来月は何をするか」
    - **場所**: Discord `#アナウンス` やフォーラムで、誰もが見られる形で公開します。
- [ ] **定例ミーティング**（担当: 全員/任意）
    - **時期**: 毎月第1●曜日 20:00〜 など
    - **内容**: 提案中の議題についての議論、交流会
    - **場所**: Discord ボイスチャンネル

### 四半期ごと（Quarterly）
- [ ] **四半期予算策定**（担当: 業務執行社員）
    - **時期**: 期末の2週間前〜
    - **内容**: 次の3ヶ月の活動予算案を作成し、DAO総会（通常決議）にかける。
- [ ] **KPT振り返り**（担当: 全員）
    - **時期**: 期末
    - **内容**: Keep（良かったこと）、Problem（課題）、Try（改善案）を出し合い、運営ルール（細則）を微修正する。

## 年次カレンダー（Annual）

### 4月（年度初め）
- **新年度キックオフ**: 今年度の方針発表、新予算の執行開始。

### 5月〜6月（決算準備）
- **計算書類の作成**: 3月決算の場合、5月末までに税務申告が必要です。税理士と連携して進めます。
- **DAO総会（決算承認）**: 作成した計算書類をDAO総会（通常決議または重要決議）で承認します。
    - **期限**: 事業年度終了後3ヶ月以内（定款第22条）。

### 10月（半期レビュー）
- **中間報告**: 上半期の振り返りと、下半期の修正計画があれば提案します。

### 3月（年度末）
- **次年度事業計画**: 来年の計画を立て始めます。
- **役員改選（必要なら）**: 業務執行社員を交代する場合、この時期に選挙（重要決議）を行うと新年度からスムーズに移行できます。

## 不定期・随時発生するタスク
- **議題提案**: 誰でもいつでも提案できます（5名の賛同が必要）。
- **トークン発行**: 新規メンバー加入時や、貢献者への報酬配布時。
- **インシデント対応**: 何かトラブルがあったら即座に動きます（[インシデント対応](./06_incident.md)参照）。
