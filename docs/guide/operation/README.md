## 1. 役割と責務（誰が何を？）

### 業務執行社員（Admin / 代表者）
法的責任を負う「運営チーム」です。
- **主な責務**:
    - 議案の形式チェックと投票設定
    - 投票結果の実行（送金、契約締結など）
    - 定期的な活動報告
    - インシデント時の緊急対応
- **権限**:
    - トレジャリーのマルチシグ署名権
    - Discord/投票ツールの管理者権限

### DAOメンバー（Member / 投票者）
議決権を持つ「参加者」です。
- **主な責務**:
    - 提案内容の確認と議論への参加
    - 投票（意思表示）
    - 不正の監視
- **権限**:
    - 議決権（提案への賛成/反対）
    - 議題提案権（一定の賛同が必要）

## 2. 定常業務カレンダー（年間スケジュール）
DAOにも「決算」や「定例会」があります。

### 毎月やること
- **月次報告**: 業務執行社員が前月の活動と会計（大まかな入出金）を報告。
- **定例ミーティング**: 必要に応じて開催（テキストチャットでも可）。

### 四半期（3ヶ月）ごと
- **予算見直し**: 次の四半期の予算を提案・承認。
- **KPT振り返り**: 運営プロセスの改善点を議論。

### 年次（1年）ごと
- **決算承認**: 事業年度終了後3ヶ月以内に、計算書類（決算書）を承認。
- **役員改選**: 必要であれば業務執行社員の再任/交代を投票で決定。
- **税務申告**: 法人税等の申告・納税（税理士へ依頼）。

## 3. 運用タスク一覧（Scan Resultsより）

| 頻度/トリガー | タスク名 | 担当 | アクション |
| :--- | :--- | :--- | :--- |
| **随時** | **議題提案** | メンバー | 指定Chでスレッド作成 → 議論 → 5名賛同でProposalへ |
| **投票開始時** | **投票告知** | 運営 | 開始前日までに全メンバーへ通知（期間・概要・URL） |
| **投票終了後** | **結果開示** | 運営 | 終了後7日以内に結果を開示し、議事録を作成 |
| **随時** | **トークン発行** | 運営 | 条件達成時に付与し、名簿を更新 |
| **随時** | **紛失再発行** | 運営 | 申請受領 → 本人確認 → 旧トークン無効化 → 再発行 |
| **緊急時** | **緊急投票** | 運営 | 20%参加・2/3賛成で即時投票開始（通常期間を短縮） |
