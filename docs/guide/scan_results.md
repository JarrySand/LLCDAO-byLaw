## 1. 概要
本ドキュメントは、`docs/guide/revamp_plan.md` の Phase 1 に基づき、定款・各規程・別紙をスキャンして抽出した「必須パラメータ（立ち上げ時決定事項）」と「運用タスク（定常業務）」の一覧です。

## 3. 運用タスク一覧 (Operational Tasks)
DAO稼働後に発生する業務タスク。

### 定期業務 (Routine)
| 頻度 | タスク名 | 担当 | 根拠条文 |
| :--- | :--- | :--- | :--- |
| **毎週** | 委任決議の執行状況報告 | 業務執行社員 | 運営規程 第8条 |
| **毎月** | 重要決議の執行状況報告 | 業務執行社員 | 運営規程 第8条 |
| **四半期** | 通常決議の執行状況報告 | 業務執行社員 | 運営規程 第8条 |
| **年次** | 計算書類の承認・開示 | 業務執行社員 | 定款 第22条 / 運営規程 第12条 (終了後90日以内) |
| **随時** | トークン発行状況の開示 | 業務執行社員 | 運営規程 第12条 (要求時速やかに) |

### イベント発生時 (On Event)
| トリガー | タスク名 | 手順概要 | 根拠条文 |
| :--- | :--- | :--- | :--- |
| **議題提案** | 提案・議論・賛同集め | 指定Chでスレッド作成→議論→5名賛同でProposalへ | 総会規程 第4条 |
| **投票開始** | 投票の告知 | 開始前日までに全メンバーへ通知 (期間/方法/議案概要) | 総会規程 第7条 |
| **投票終了** | 結果の開示・記録 | 終了後7日以内に結果を開示、議事録作成 | 総会規程 第8条 |
| **トークン発行** | トークン付与・名簿更新 | 条件達成時に付与、社員名簿(定款別紙)を自動/手動更新 | トークン規程 第13条等 |
| **トークン紛失** | 再発行手続 | 申請受領→本人確認→無効化→再発行 (5-10営業日以内) | トークン規程 第13条 |
| **執行困難** | 執行停止・報告 | 困難事由発生時、遅滞なく報告し再審議を求める | 運営規程 第9条 |
| **緊急事態** | 緊急投票 | 20%参加・2/3賛成で即時投票開始 | 総会規程 第7条 |
| **重大事故** | 事後開示 | 破産・処分・災害等は遅滞なく開示 | 運営規程 第11条 |
| **請求受領** | 請求通知・対応 | 第三者請求時、業務執行社員へ通知し措置を講じる | 定款 第20条 |
| **利益相反取引** | 事前承認決議 | 業務執行社員とDAOの取引等は総会承認が必要 | 定款 第13条 |
| **社員権譲渡** | 承認不要の確認 | トークン移転を伴う譲渡は原則社員承諾不要 | 定款 第8条 |

## 4. 特別な権利行使・手続き
通常の業務フローとは異なる、特別な権限行使のケース。

| 権利/手続名 | 行使者 | 条件・期限 | 根拠条文 |
| :--- | :--- | :--- | :--- |
| **社員同意否認** | 社員過半数 | 法令違反等に該当時、告知後一定期間内に否認可 | 定款 第18条 |
| **業務執行拒否権** | 業務執行社員 | 法令違反等に該当時、発覚後一定期間内に差止可 | 定款 第19条 / 運営規程 第2条 |
| **細則制定** | 業務執行社員 | 委任範囲内での制定可、要通常決議承認 | 運営規程 第14条 |
