# ガイド概要

このガイドは、合同会社型DAO定款・規程ひな形を**理解し、活用する**ための資料です。

---

## ⚠️ このひな形のスコープについて

**本ひな形は「DAOが実社会とつながるためのツール」です。**

DAOが日本法に準拠しながら、契約の主体となり、銀行口座を持ち、有限責任の恩恵を受けるための「法人という器」を整備することが目的です。

### スコープに含まれるもの ✅
- 合同会社型DAOの定款・規程のひな形
- 法人設立・運営の実務手順
- 日本法に準拠した適法な運用のためのガイドライン

### スコープに含まれないもの ❌
- **DAOのデザイン**（どんな目的・ミッションを持つべきか）
- **「良いDAO」の定義や評価**（どのようなガバナンスが理想か）
- **コミュニティ運営のノウハウ**（メンバーのエンゲージメントをどう高めるか）
- **トークノミクス設計**（どのような経済設計が成功するか）

本ひな形は、あなたのDAOが「どうあるべきか」を定義するものではありません。あなたのDAOが「実社会で機能するため」の法的な器と手続きを提供するものです。

DAOの目的・ビジョン・コミュニティのあり方は、各DAOが自ら定めるべきものです。本ひな形は、その多様なDAOが日本法の下で活動するための共通基盤として設計されています。

## 🎯 このガイドの使い方

### はじめての方

1. まず [設計思想（context）](./context.md) で、このひな形が解決しようとしている課題と設計原則を理解する
2. [構成/参照/版管理（structure）](./structure.md) で、ドキュメント間の関係を把握する
3. 定款・規程の本文を読む

### DAOを立ち上げたい方

1. [立上げガイド](./setup/README.md) に沿って、必要な決定事項を整理する
2. ひな形をフォークまたはコピーし、パラメータを自分たちのDAOに合わせてカスタマイズする
3. 専門家（弁護士・司法書士等）に相談のうえ、法人設立を進める

### 既存DAOの運営者

1. [運用ガイド](./operation/README.md) を参考に、日常業務のフローを整備する
2. 定常業務カレンダーをもとに、月次・四半期のタスクを管理する
