## docs/guide/

運用ガイド・手順・ベストプラクティスなど、定款・各規程の解釈や実装支援に役立つガイドを整理します。

- 目次
  - [context.md](./context.md): 設計思想（共通原則・前提・トレードオフ）
  - [structure.md](./structure.md): 定款と規程の構成・参照関係・版管理
  - [operation.md](./operation.md): 実運用手順・チェックリスト・ベストプラクティス

注意: ここは参考資料ですが、`docs/guide/` 配下（context/structure/operation）は公開コメントのレビュー対象に含みます。Issue/PR から直接ご提案ください。
