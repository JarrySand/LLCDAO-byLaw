## 7. ツール選定（投票/議事録/ダッシュボード/保管）

TL;DR（5分で決める）
- 投票/署名/議事録/ダッシュボード/文書保管の標準ツールを選定
- バックアップ/移行手段と所有権（アカウント/課金）を明確化
- 初期テンプレとアクセス配布

決定パラメータ
- オンチェーン/オフチェーンの住み分け
- 権限モデルと監査ログ
- 保管と公開の分離（機微情報）

チェックリスト
- [ ] 各ツールの初期設定・権限付与
- [ ] モニタリング/可観測性の設定
- [ ] エクスポート/バックアップの検証

---

サンプル初期設定（例）
- 投票: Snapshot、署名: Safe、議事録: GitHub、保管: Git + Notion公開
- 可視化: GitHub Projects/Sheets/Dune いずれか
- バックアップ: 月次エクスポート、所有者=法人アカウント

1ページ総まとめ（章内）
- 所有権と請求先を個人→法人へ統一し、引継ぎリスクを排除
