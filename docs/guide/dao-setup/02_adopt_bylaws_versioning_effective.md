## 2. 規程の採択・版管理と発効準備

TL;DR（5分で実行）
- 採択手続（誰が・どう決めるか）を確定
- 初期版（v2.x）の採択日・発効日・タグ名を決定
- 公開場所とアーカイブ運用を設定

決定パラメータ
- 採択権限/総会の要否
- 発効日と移行期間
- 公開コメントの取り扱い

チェックリスト
- [ ] 初期版のタグ/アーカイブが用意
- [ ] 採択議事録/決議の証跡
- [ ] 周知計画（誰に何をいつ）

---

サンプル初期設定（例）
- 採択権限: 初回は創設メンバー合意→初回総会で追認
- 発効: 採択の翌営業日、移行期間2週間
- 公開: GitHub公開＋Notion同期、タグ `bylaws-v2.1-initial`
