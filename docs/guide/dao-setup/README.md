## DAO立ち上げガイド（Setup Guide）

このガイドは、雛形（定款/各規程/別紙）に「従って」DAOを新規に立ち上げる際に、何を決め・準備し・実行すべきかを網羅的に示します。各章は TL;DR（5分）とチェックリスト中心で、決定すべきパラメータを明示します。

### Start Here（フェーズ別）
- フェーズ0: 目的/範囲/法域の確定（ステークホルダー合意）
- フェーズ1: 規程の採択/発効準備
- フェーズ2: トークン設計・権限スキーム確定
- フェーズ3: トレジャリー・会計・承認フロー整備
- フェーズ4: 運用体制/ツール/セキュリティ準備

### 章立て
1. 目的・範囲・法域の確定（ガバナンス方針）
2. 規程の採択・版管理と発効準備
3. ガバナンストークン/社員権トークン設計と権限モデル
4. 投票方式・閾値・定足数・委任ポリシーの決定
5. トレジャリー設計（財布構成/承認しきい値/可視化）
6. 役割・責任・連絡チャネル・RACIの確定
7. ツール選定（投票/議事録/ダッシュボード/保管）
8. セキュリティ/リスク/コンプライアンス準備
9. ローンチ手順（初回総会/提案/資金初期化）

関連: 運用開始後は `docs/guide/dao-runbook/` を参照。

---

### 1ページ総まとめ（まずはここだけ埋める）
- 目的/範囲/法域: 何を達成し、どこで運用するか
- 規程採択: 採択日・発効日・公開先・タグ名
- トークン/権限: 種別・供給・権限・委任の基本方針
- 投票要件: 重要度別の閾値/定足数/期間
- トレジャリー: 財布構成・署名者・しきい値・ダッシュボード
- 役割/RACI: 主要ロール/バックアップ/承認者/連絡網
- ツール: 投票/議事録/保管/可視化の標準
- セキュリティ/コンプラ: 初動SOP・鍵管理・公開方針
- ローンチ: 初回総会日程・初期提案・初期資金
