## 8. セキュリティ/リスク/コンプライアンス準備

TL;DR（5分で実行）
- 重大度マトリクスと初動SOPを確定
- 鍵管理/署名ポリシー/ローテーション手順を合意
- 公開範囲/個人情報/秘密情報の取り扱い基準を決定

決定パラメータ
- 重要連絡網/オンコール
- バグバウンティ/責任ある開示方針の有無
- 法務・税務・会計の相談窓口

チェックリスト
- [ ] インシデント演習の計画
- [ ] ログと証跡の保持期間
- [ ] データ分類とアクセス制御

---

サンプル初期設定（例）
- 初動SOP: 凍結→告知→原因分析→恒久対策→公開
- 鍵管理: ハードウェアキー必須、ローテーション半期、退任時即時失効
- 公開方針: 提案/決議/資金は公開、個人情報は匿名化

1ページ総まとめ（章内）
- リスク登録票を作成し、重大度×確率で優先度管理
