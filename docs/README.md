## docs/

このディレクトリは、定款・各規程そのものではなく、周辺ドキュメント（設計思想、構成、運用）を整理するための場所です。

- ガイド（`guide/`）: 設計思想・構成・運用ガイド
  - [guide/context.md](./guide/context.md) — 定款・規程の設計思想（共通原則/前提/トレードオフ）
  - [guide/structure.md](./guide/structure.md) — 定款と規程の構成・参照関係・版管理
  - [guide/operation.md](./guide/operation.md) — 実運用手順・チェックリスト
注意: 公開コメントの対象は、リポジトリ直下・`appendices/` の主要文書に加えて、`docs/guide/` 配下（context/structure/operation）も含みます。

### 構成
```
docs/
├── README.md       # この説明書
├── guide/          # 運用ガイド・手順・ベストプラクティス
│   ├── context.md  # 設計思想（共通原則）
│   ├── structure.md# 構成/参照/版管理
│   └── operation.md# 実運用
```

### 置き場所の目安
- `guide/context.md`: 共同体として共有する設計原則・前提・トレードオフ
- `guide/structure.md`: ファイル/条文の配置、参照関係、版管理方針
- `guide/operation.md`: 手順、チェックリスト、実務的な注意点
