# 緊急時対応規定まとめ

**作成日**: 2024年  
**目的**: ver2.0型定款における緊急時対応メカニズムの整理

---

## 概要

ver2.0型の規程体系では、緊急時対応は**定款ではなく下位規程（DAO総会規程・運営規程）で対応する設計**になっています。これはmacverの但書（定款レベル）とは異なるアプローチです。

---

## 緊急時対応規定一覧

### 1. DAO総会規程ver2.1

#### 第7条第2項（緊急投票）

```
2. 前項に関わらず、緊急に決議が必要である場合は、議決権保有者がもつ
   総議決権数の20%以上の投票参加により、3分の2の賛成をもって、
   直ちに投票を開始することができる。
```

| 項目 | 内容 |
|------|------|
| **発動要件** | 緊急に決議が必要である場合 |
| **参加要件** | 総議決権数の20%以上の投票参加 |
| **可決要件** | 3分の2の賛成 |
| **効果** | 通常の投票期間を経ずに即時投票開始 |

#### 第14条（特例）

```
やむを得ない理由により第6条に基づく決議がなされない場合は、
業務執行社員は決議がなされるまでの間、当該決議までに議決された事項に準じ
収入支出することができる。ただし、重要な財産の処分及び譲受け並びに
多額の借財をすることはできない。
```

| 項目 | 内容 |
|------|------|
| **発動要件** | やむを得ない理由により決議がなされない場合 |
| **権限範囲** | 過去の決議事項に準じた収入支出 |
| **制限事項** | 重要な財産の処分・譲受け・多額の借財は不可 |

---

### 2. 運営規程ver2.1

#### 第7条第1項（業務執行の再委任）

```
1. 業務執行社員は、DAO総会の委任決議に基づく業務執行を行うに際しては、
   以下の範囲で業務執行の権限を第三者に再委任することができる。
   - 日常的業務の執行（月額100万円以下の支出を伴う業務）
   - 定型的契約の締結（標準契約書に基づく契約）
   - 緊急時対応（事業継続に必要な応急措置）
```

| 項目 | 内容 |
|------|------|
| **位置づけ** | 業務執行社員から第三者への再委任範囲 |
| **緊急時対応の定義** | 事業継続に必要な応急措置 |
| **前提** | DAO総会の委任決議が存在すること |

---

### 3. トレジャリー管理規程v1.0（作成中）

#### 異常発生時の対応

| 項目 | 内容 |
|------|------|
| **緊急DAO総会** | 異常発生時は必要に応じて緊急DAO総会を開催 |
| **一時的措置** | 財務担当業務執行社員の判断により一時的な取引停止等の措置 |
| **裁量範囲** | 緊急時の応急措置費用（月額10万円以下） |

## macver但書との比較

### macver第6条第1項但書

```
但し、合理的必要性があって、業務執行社員の過半数が同意した場合は、
業務執行社員が定める方法で発行されるものを、「社員権トークン」と
扱って良いものとする。
```

### 比較表

| 観点 | macver但書 | ver2.0型（規程対応） |
|------|-----------|---------------------|
| **規定レベル** | 定款（変更困難） | 下位規程（変更容易） |
| **発動要件** | 業務執行社員過半数同意 | 20%参加・2/3賛成 |
| **手続きの明確性** | 「合理的必要性」は曖昧 | 要件が数値化され明確 |
| **民主性** | 業務執行社員の判断 | 議決権保有者の投票 |
| **透明性** | 記録義務なし | 投票記録が残る |
| **濫用リスク** | 恣意的運用の余地あり | セーフガードあり |

---

## 結論

**ver2.0型でも緊急時対応は十分に可能**であり、macverの但書（第6条第1項但書）は不要と考えられます。

### ver2.0型＋下位規程の利点

1. **手続きが明確**（20%参加・2/3賛成という数値基準）
2. **民主的**（業務執行社員の一存ではない）
3. **透明性が高い**（投票記録が残る）
4. **柔軟性**（規程レベルなので改定が容易）
5. **セーフガード**（制限事項が明記されている）

### 推奨

論点10（代替トークン発行方法・緊急時但書）については、**ver2.0型（但書なし）を推奨**。緊急時はDAO総会規程第7条第2項の緊急投票制度で対応可能。
