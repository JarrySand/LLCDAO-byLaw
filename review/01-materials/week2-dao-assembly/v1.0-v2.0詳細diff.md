## 🔄 第5条（免責事項 → 社員同意手続き）

```diff
- 第５条（免責事項）
- 前条に従い社員権トークンホルダーから提案がなされた議題について、必要に応じて当該提案者及び賛同者の意見を聞いた上で、業務執行社員は、第１１条に定める免責事項に該当する場合は、DAO総会で決議することを拒否するものとする。...
+ 第5条（社員同意手続き）
+ 会社法その他の法令により社員による同意又は決定等が定められている事項については、本定款第18条（社員の同意）の規定に従うものとする。
```

— v1.0の免責に関する規定を削除し、会社法準拠の「社員同意」への準用規定に置換。免責の位置づけは後述の拒否権規定に整理。

## 🔄 第6条（決議事項の区分と要件）

```diff
- （１）レベル5：DAO総会特別投票
+ （1）重要決議
   - 【定款の変更、役員選解任、配当等】の最重要事項
- （２）レベル4：DAO総会投票
+ （2）通常決議
   - 【各種規程の制定/改廃、予算承認等】
- （３）レベル3：DAO総会臨時投票
+ （3）委任決議
   - 【業務執行社員への委任事項のうち信任が必要なもの】

- ② レベル５：総議決権数の20%を有する者が投票し、過半数の賛成。
+ 2. 重要決議：総議決権数の20%以上の投票参加、投票者過半数。
- ② レベル４：総議決権数の5%を有する者が投票し、過半数の賛成。
+ 2. 通常決議：総議決権数の5%以上の投票参加、投票者過半数。
- ② レベル３：定足数を設けず、投票過半数で承認。
+ 2. 委任決議：定足数なし、投票過半数で承認。
```
