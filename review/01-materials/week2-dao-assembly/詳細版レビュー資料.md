### 2. 決議制度の法的妥当性【15分】

#### 📋 3段階決議制の法理論的根拠
**従来の5段階制**: レベル1-5の複雑な階層
**新3段階制**: 重要・通常・委任の明確な分類

**重要決議の要件**:
- 対象: 会社存続・分散性維持に重大影響を与える事項
- 定足数: 総議決権数の20%以上
- 決議要件: 投票者の過半数賛成
- **具体的事項**:
  - 定款変更（会社法第637条準用）
  - 業務執行社員選任・解任（会社法第598条）
  - 利益配当・残余財産分配（会社法第621条）
  - 利益相反取引承認（会社法第595条）
  - 支配人選任・解任（会社法第613条）
  - 社員退社（会社法第859条）
  - 清算人解任（会社法第647条）
  - 組織変更・吸収合併（会社法第793条等）

**通常決議の要件**:
- 対象: DAO運営に重要で総会審議が必要な事項
- 定足数: 総議決権数の5%以上
- 決議要件: 投票者の過半数賛成
- **具体的事項**:
  - 各種規程制定・改廃
  - 年次予算承認
  - 重要業務執行方針の決定
  - 委任事項の範囲決定
  - 業務執行社員への委任決議

**委任決議の要件**:
- 対象: 委任事項について信を問う場合
- 定足数: 制限なし
- 決議要件: 投票者の過半数賛成
- **具体的事項**:
  - 委任業務の執行状況報告
  - 委任権限の継続可否
  - 緊急時業務執行の追認

**検討課題**:
- [ ] **段階設定**: 3段階の区分基準は法的に合理的か？
- [ ] **定足数**: 20%・5%・なしの設定根拠は適切か？
- [ ] **決議要件**: 全て過半数賛成で統一する合理性は？
- [ ] **会社法整合**: 株主総会制度との比較で適法性は？

#### 📋 ガバナンストークンホルダーの議決権
**参加者拡大**: 社員権トークンホルダー + ガバナンストークンホルダー

**議決権計算** (技術仕様補足):
```
総議決権数 = 社員権トークン保有数 + ガバナンストークン保有数
※リワードトークンは議決権を持たない
```

**法的位置づけ**:
- **社員権トークンホルダー**: 社員資格 + 議決権
- **ガバナンストークンホルダー**: 議決権のみ（社員資格なし）

**検討課題**:
- [ ] **法的根拠**: 社員でない者の議決権行使の法的根拠は？
- [ ] **会社法整合**: 出資と議決権の分離は会社法上適法か？
- [ ] **利益相反**: 社員と非社員の利益が対立する場合の処理は？
- [ ] **権利保護**: 各トークンホルダーの権利保護は十分か？

#### 📋 業務執行社員への権限集中
**投票管理権限**: 
- 第8条: DAO総会投票管理は業務執行社員
- 第8条第2項: 議長は代表社員又は業務執行社員互選

**委任権限**:
- 第9条: 通常決議により重要決議以外を委任可能

**拒否権関与**:
